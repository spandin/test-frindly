## Установка

Инструкции по установке и запуску проекта:

1.  Клонируйте репозиторий:

    ```bash
    git clone [URL вашего репозитория]
    ```

2.  Перейдите в директорию проекта:

    ```bash
    cd [название вашей папки проекта]
    ```

3.  Установите зависимости:

    ```bash
    npm install
    ```

4.  Запустите проект:

    ```bash
    npm run dev
    ```
